⚙️ Installation Instructions

Follow these steps to install and activate your BTC bot system.

  1. Download the full bot system from the button below.
  2. Unzip the package into a folder on your computer.
  3. Install Python 3.10+ if you haven’t already.
  4. Install the required libraries using: 
    pip install -r requirements.txt
  5. Open .env file and paste your Kraken API credentials.
  6. Launch the bot using the included run_bot.bat file.
